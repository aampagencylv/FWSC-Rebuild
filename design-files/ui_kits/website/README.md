# FWSC Website UI kit

Public website for the Florida Water Sports Coalition. Three screens, clickable via header nav:

- **Home** — masthead, hero, advisory banner, program cards, membership band
- **Advisories** — tabbed list of active waterway advisories with status badges
- **Directory** — certified operator directory with region/vessel filters

Composed from the design-system components (`window.FWSCDesignSystem_db6c3e`). Open `index.html`.
No product source existed; this kit *establishes* the website design per the brand foundations.
